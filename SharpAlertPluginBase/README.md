Hello there!

This is a package for developers to develop plugins for SharpAlert. As of now, this package serves as a simple way to receive alerts before they are relayed into the rest of the program, but this will be expanded upon in the future.

Implement IPlugHandler from this package in your application to get started!

https://bunnytub.com/SharpAlert (plugin implementation will be available in SharpAlert v16 and onwards)
